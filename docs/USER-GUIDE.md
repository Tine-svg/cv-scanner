## การเข้าใช้งานครั้งแรก

### 🔐 Login

1. เปิด URL ของระบบ (ที่ admin แจ้งให้)
2. กรอก **Username** และ **Password** ที่ admin ให้มา
3. กด **"เข้าสู่ระบบ"**

### 🔑 เปลี่ยนรหัสผ่านครั้งแรก

แนะนำให้เปลี่ยนรหัสผ่านทันทีหลัง login ครั้งแรก:

1. มุมขวาบน → กดไอคอน **🔒 (กุญแจ)**
2. กรอก:
   - **รหัสผ่านเดิม:** ที่ admin ให้
   - **รหัสผ่านใหม่:** ตั้งใหม่ (≥ 4 ตัว)
   - **ยืนยันรหัสใหม่:** กรอกซ้ำ
3. กด **บันทึก** → เห็น "✓ เปลี่ยนรหัสผ่านสำเร็จ"

## อัพโหลด CV

### วิธีที่ 1: ลากไฟล์มาวาง

1. ลากไฟล์ PDF หรือ Word จาก folder ในเครื่อง
2. วางในกล่องประที่ "คลิกหรือลากไฟล์มาวางที่นี่"
3. ระบบจะเริ่มสแกนทันที

### วิธีที่ 2: เลือกผ่าน dialog

1. **คลิกที่กล่องอัพโหลด** → dialog เลือกไฟล์เปิดขึ้น
2. เลือก **1 ไฟล์ หรือหลายไฟล์** (กด Ctrl + คลิก / Shift + คลิก)
3. กด **Open**

### 📋 ตรวจสอบและแก้ไขก่อนบันทึก

หลังสแกน — ระบบจะแสดงตารางพร้อมข้อมูลที่ดึงมาได้:

| ช่อง | ความหมาย |
|---|---|
| ✅ | ติ๊ก = บันทึก, ไม่ติ๊ก = ข้าม |
| ชื่อไฟล์ | ชื่อไฟล์เดิม |
| ชื่อ-นามสกุล | ดึงจาก CV |
| ตำแหน่ง | ตำแหน่งที่สมัคร |
| Email / เบอร์โทร | ข้อมูลติดต่อ |
| อายุ / เพศ | ถ้าระบุไว้ใน CV |
| โน้ต | เพิ่มได้ตอนนี้เลย |

**สามารถแก้ไขได้ทุกช่อง** — คลิกที่ช่องแล้วพิมพ์

### 🔄 ตรวจไฟล์ซ้ำ

ระบบใช้ **SHA-256 hash** เช็คว่าไฟล์ซ้ำกับที่เคยอัพแล้วไหม:

- ถ้าซ้ำกับใน database → แสดง badge **"ซ้ำ"** สีเหลือง
- มี dropdown ให้เลือก:
  - **ข้าม** — ไม่บันทึก
  - **แทนที่ของเก่า** — อัพเดตข้อมูลของ record เดิม
  - **เพิ่มเป็นรายการใหม่** — บันทึกแยกต่างหาก

### ✅ ยืนยันบันทึก

กด **"บันทึก"** มุมล่างขวา → ข้อมูลทั้งหมดที่ติ๊กไว้จะถูก sync ไป Supabase

## ค้นหาผู้สมัคร

### 🔍 ช่องค้นหา

1. ตารางด้านล่าง มีช่องค้นหา **"ค้นหาด้วยชื่อ, ตำแหน่ง, email, เบอร์, อายุ, เพศ, โน้ต, วันที่..."**
2. พิมพ์คำที่ต้องการ → ตารางกรองทันที (real-time)
3. คำที่ตรงกันจะถูก **highlight สีเหลือง** ในแถว

### 💡 เคล็ดลับการค้นหา

- พิมพ์ส่วนหนึ่งของคำได้ — ไม่ต้องเต็ม เช่น พิมพ์ `dev` จะเจอทั้ง "Developer" และ "DevOps"
- ค้นด้วยปี (พ.ศ.) เช่น `2569` เพื่อหาคนที่สมัครมาในปี 2569
- ค้นด้วยชื่อคนที่อัพโหลด — เช่น `jane` จะเจอ CV ที่ Jane เป็นคนเพิ่ม

## แก้ไขข้อมูล

หลังบันทึกแล้ว — ยังแก้ไขข้อมูลได้:

### ✏️ แก้ข้อมูลทั่วไป

1. ในตาราง → คอลัมน์ **Actions** → กด **"แก้ไข"**
2. Card เปิดขึ้นมาด้านบน — แก้ฟิลด์ที่ต้องการ
3. กด **"บันทึก"** → ข้อมูลอัพเดต sync ไป cloud

### 📝 แก้โน้ตเร็วๆ

ในตาราง — ช่อง **"โน้ต"** เป็น input box คลิกได้เลย:

1. คลิกที่โน้ตในแถวที่ต้องการ
2. พิมพ์ → กด **Tab** หรือคลิกที่อื่น
3. ระบบ auto-save ทันที

### 🗑️ ลบ record

1. กด **"ลบ"** ในแถวนั้น
2. มี dialog ยืนยัน → กด OK
3. record + ไฟล์ PDF จะถูกลบจาก cloud

⚠️ **ลบแล้วเอากลับไม่ได้** — ระวัง!

## ดู / ดาวน์โหลดไฟล์ต้นฉบับ

ในตารางผลลัพธ์ — แต่ละแถวมีปุ่ม **"ดู"** สีฟ้า:

1. กดปุ่ม **"ดู"**
2. Modal เปิดขึ้นมา:
   - **PDF** — แสดง preview เต็มหน้า ดูได้เลย
   - **Word** — แสดงข้อความว่าให้ดาวน์โหลดเปิดด้วย Microsoft Word
3. ปุ่ม **"⬇ ดาวน์โหลด"** มุมขวาบน — โหลดไฟล์ลงเครื่อง

ปิด modal: กด **×** หรือ **ESC**

## เพิ่มโน้ต

### โน้ตสามารถเพิ่มได้ 3 จุด:

**1. ตอนอัพโหลด** (ใน batch preview)
- ในตารางก่อนบันทึก → ช่อง "โน้ต" → พิมพ์เลย

**2. ตอนแก้ไข**
- กด "แก้ไข" → มีช่อง "โน้ต" ในฟอร์ม

**3. ในตารางผลลัพธ์ (เร็วสุด)**
- คลิกในช่อง "โน้ต" ของแถวนั้น → พิมพ์ → คลิกที่อื่น → save อัตโนมัติ

### 💡 ใช้โน้ตยังไงให้มีประโยชน์

- บันทึกผลสัมภาษณ์: "ผ่านรอบแรก, สัมภาษณ์ 12 พ.ค."
- Tag คุณสมบัติ: "Senior, React, 5 ปี"
- Note ข้อสังเกต: "ภาษาอังกฤษดี, ขอเงินเดือนสูง"

โน้ตค้นหาได้จากช่อง search ด้วย!

## โปรไฟล์ของคุณ

ดูข้อมูลโปรไฟล์ของคุณ — มุมขวาบน:

```
🟣 jane   🔒    🚪
ชื่อ      เปลี่ยน  ออก
         รหัส
```

### 🔑 เปลี่ยนรหัสผ่าน
ไอคอน **🔒** → กรอก: รหัสเดิม + รหัสใหม่ + ยืนยัน

### 🚪 Logout
ไอคอน **🚪** → ยืนยัน → กลับไปหน้า Login

### 📸 ดูข้อมูลตัวเอง / เปลี่ยนรูป
- ใน CV ที่คุณอัพโหลด — คอลัมน์ "โดย" จะมี pill ของคุณ
- คลิก pill → popup แสดงข้อมูล (แต่ **เปลี่ยนรูปต้องให้ admin แก้ให้**)

## Troubleshooting

### เข้าระบบไม่ได้

| อาการ | สาเหตุ | วิธีแก้ |
|---|---|---|
| "username หรือ password ไม่ถูกต้อง" | พิมพ์ผิด | ลองพิมพ์ใหม่ ระวัง Caps Lock |
| ลืมรหัสผ่าน | จำไม่ได้ | บอก admin ให้ reset ให้ |
| ไม่มี account | ยังไม่ได้สมัคร | ขอ admin สร้าง account ให้ |

### อัพโหลดไม่ได้

| อาการ | สาเหตุ | วิธีแก้ |
|---|---|---|
| "ไฟล์ใหญ่เกิน 10MB" | ไฟล์ใหญ่ | บีบอัด PDF หรือลบรูปใน CV |
| สแกนแล้วไม่เจอชื่อ | format CV แปลก | แก้ในฟอร์มก่อนบันทึก |
| "บันทึกไม่ได้: ..." | network / RLS | รีเฟรชหน้า แล้วลองใหม่ |

### ตารางไม่อัพเดต

- กด **F5** เพื่อโหลดใหม่
- ถ้ายังไม่อัพเดต → ตรวจ internet
- ถ้ายังไม่ใช่ → บอก admin

### ปุ่ม "ดู" PDF ไม่ทำงาน

- รอ pdf.js โหลดเสร็จ (~2 วิ)
- ถ้ายังไม่ได้ → กด **"⬇ ดาวน์โหลด"** แล้วเปิดในเครื่องแทน

## 💬 ติดต่อ Admin

ถ้ามีปัญหาที่แก้เองไม่ได้ — บอก admin พร้อมข้อมูล:

1. **ทำอะไรอยู่** ตอนเกิดปัญหา
2. **ภาพหน้าจอ** ที่เกิด error
3. **เวลา** ที่เกิดเหตุ (ประมาณ)

จะช่วยแก้ได้เร็วขึ้นมาก 🙏
