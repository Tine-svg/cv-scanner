เชื่อมต่อ Supabase

วาง Project URL และ anon key เพื่อเก็บข้อมูลและไฟล์บน Supabase ดูวิธีหา ↓

  1. เปิด supabase.com/dashboard → เข้า project
  2. เมนูซ้าย → ⚙ Settings → API
  3. คัดลอก Project URL (เช่น https://xxx.supabase.co) → วางในช่องแรก
  4. คัดลอก anon public key (ยาวๆ ขึ้นต้น eyJ...) → วางในช่องที่สอง
  5. ก่อนใช้ — ต้องสร้าง table และ storage bucket ตามคู่มือ supabase-setup-guide.md
